
# 📞 SentinelCall AI

### AI Powered Scam Call Detection System

🚨 **SentinelCall AI** is an intelligent system designed to protect users from **fraud and scam calls in real time**.
The system uses **Artificial Intelligence, Speech Recognition, and NLP** to analyze phone conversations and detect suspicious behavior instantly.

The goal of this project is to **reduce digital fraud and protect people from scam calls** by providing instant alerts and fraud risk scores.

---

# 🧠 Project Idea

With the rapid increase in **online fraud and scam calls**, many people fall victim to fake bank calls, OTP scams, lottery scams, and phishing attacks.

SentinelCall AI solves this problem by:

• Converting call audio into text using **Speech Recognition**
• Analyzing conversations using **NLP models**
• Detecting fraud keywords and suspicious intent
• Generating a **fraud risk score**
• Alerting the user if the call is potentially dangerous
